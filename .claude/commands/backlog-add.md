Add a new item to the SafePaste feature backlog.

1. Read docs/backlog.md to understand current priorities.

2. Ask the user (or determine from context):
   - What is the feature/improvement?
   - Why is it needed?
   - What priority? (high / medium / low)
   - Any dependencies or blockers?

3. Add the item to the appropriate priority section in docs/backlog.md.

4. If this item relates to a known risk, cross-reference it in
   docs/project-state/known-risks.md.

5. If this item relates to a roadmap phase, cross-reference it in
   docs/roadmap/sdk-roadmap.md.
